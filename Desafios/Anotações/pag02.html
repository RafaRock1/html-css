Trabalho com Listas

Listas Ordenadas

Para listas ordenadas utilize a Tag <OL>
Também temos os type que fica dentro da tag <OL> ex:
<OL type="1">
Existe os types ( 1 | A | a | I | i )

Para criar um item dentro da listas utilize a Tag <LI> dentro da Tag <OL> exemplo:

<OL type="1">

  1. <LI>acordar
  2. <LI>tomas café
  3. <LI>escovar os dentes
  4. <LI>ligar pra mãe
  5. <LI>almoçar

</OL>

<UL type="disc">
