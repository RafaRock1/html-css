Usando links externos

Sobreposição de Site

Você usa a Tag <A href="(LINK)">INSTAGRAM</A>

Ex: Você pode acessar meu INSTAGRAM

Site em Nova Aba

Você usa a Tag 

            <A href="(LINK)" target="_blank" rel="external">TIK-TOK</A>

(target="_blank") é o comando para abrir o site em uma nova página.
(rel="external") é o comando que indicar o nome da pagina para o algoritimo do google identificar.

Ex: Você pode acessar meu TIK-TOK


Usando Links Internos

Links internos são os documentos HTML que estão na mesma pasta.

Link em Pasta Anterior

Para acessar um link que está em uma pasta anterior, usa (../) que signicica voltar uma pasta.

Ex: Aperte aqui para ir para PagTest
