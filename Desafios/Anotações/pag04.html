Imagem Dinâmica

Tente abrir esse site em vários dispositivos diferentes oou simplemente aumente ou diminua o tamanho do seu navegador.
Usando a Tag <PICTURE> e dentro dela <source:media:type>
Ex:

            
</picture>
    <source media="(max-width: 730px)" srcset="Imagens/foto-p.png" 
    type="image/png">
            
    <source media="(max-width: 1030px)" srcset="Imagens/foto-m.png" 
    type="image/png">
            
    <img src="Imagens/foto-g.png" alt="Imagem flexível">
</picture>

Reproduzindo Áudio

Vamos aprender a reproduzir áudios em HTML.
Usando a Tag <audio>

Primeira Forma

            
        <audio src="(ARQUIVO)" controls loop ></audio>

(controls) é o comando para aparecer o controle.
(loop) é o comando pra a música se repetir até darmos pause.
(Formatos Compativeis: MP3, OGG, WAV)

Segunda forma

            
        <audio preload="metadata" controls loop>
        <source src="midia/musica01.mp3" type="audio/mpeg">
        </audio>
