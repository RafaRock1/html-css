Principais formatações

Negrito / Destaque

Nesta frase, temos um - termo em destaque - usnado tag <STRONG> (semântica)

Nessa frase, temos um - termo em negrito - usando tag <B> (não semântica)

Itálico / Ênfase

Nesta frase, temos um - termo em ênfase - usando a tag <EM> (semântica)

Nesta frase, temos um - termo em itálico - usando a tag <I> (não semântica)

Endereço

Podemos criar um endereço usando a Tag <ADDRESS>

- Rua Júdia do Melco,700 -

Texto mascado

Podemos criar um - texto mascado - usando a tag <MARK>

Texto grande / Texto pequeno

Nesta frase, temos um - texto pequeno - usando a tag <SMALL>
Esse texto tem o significado de algo implícito (semântica)

Nesta frase, temos um - texto grande - usando a tag <BIG> (está obsoleto / em des-uso)

Texto deletado

Podemos mascar um - texto como excluído - para indicar que ele deve ser lido, mas não considerado. Tag <DEL>

Inserido / Sublinhado

Podemos mascar um - texto como inserido - para dar uma ênfase e indicar que ele foi adicionado depois. Tag <INS> (semântica)

Existe também o - sublinhado - com a tag <U> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo: x20+3. Tag <SUP>

Texto subscrito

Para inserir coisas do tipo: H2O. Tag <SUB>
